What is this template

The template is a framework for building hybrid and responsive email newsletters. It consist of 3 files:

  1. HTML file containing the HTML markup and content
  2. framework.less containing all calculations for basic things like collumn widths etc
  3. styles.less contianing definitions for visual styling

How to use this template

Start building your own email template with the snippets provided in the 'Snippets' directory, or just choose an example email in the 'Examples' directory. Make sure that framework.css and styles.css are correctl;y linked.

For further documentation, have a look at the documentation on Confluence.
